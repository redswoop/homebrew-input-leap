# homebrew-input-leap

Homebrew tap for [InputLeap](https://github.com/input-leap/input-leap), an open-source KVM (keyboard/video/mouse) sharing tool.

## Install

```
brew tap redswoop/input-leap
brew install input-leap
```

## Notes

- Builds InputLeap v3.0.3 from source with Qt6
- Requires Accessibility permissions on macOS — open the app once to register
